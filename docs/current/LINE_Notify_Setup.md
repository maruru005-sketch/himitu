# LINE Notify 連携設定手順

DayTrade Terminal から LINE にアラート通知を送信するためには、LINE Notify のアクセストークンを取得し、システムに設定する必要があります。

## 1. LINE Notify トークンの取得

1. **LINE Notify 公式サイトにアクセス**
   [https://notify-bot.line.me/ja/](https://notify-bot.line.me/ja/) にアクセスし、普段お使いのLINEアカウントでログインします。
   
2. **マイページを開く**
   右上のユーザー名をクリックし、「マイページ」を選択します。

3. **トークンを発行**
   ページ下部にある「トークンを発行する」ボタンをクリックします。
   - **トークン名**: 通知時に表示される名前です（例: `DayTrade Alert`）。
   - **通知先**: 「1:1でLINE Notifyから通知を受け取る」を選択するか、専用のグループを作成してそれを選択します。

4. **トークンをコピー**
   発行されたトークン（英数字の長い文字列）が表示されます。**この画面を閉じると二度と表示されない**ので、確実にコピーして手元のメモ帳等に控えてください。

## 2. システムへの設定

1. アプリケーションの右上の「設定（⚙️）」ボタンをクリックします（機能実装後）。
2. 表示されるモーダル内の「LINE Notify トークン」の入力欄に、先ほどコピーしたトークンを貼り付けます。
3. 「保存」をクリックします。

これで設定は完了です。損切り・利確ラインへの接近時にブラウザ通知と併せてLINEにもメッセージが送信されるようになります。

> [!WARNING]
> LINE Notify トークンはパスワードと同等の機密情報です。他人に教えたり、公開される場所（GitHub等）にアップロードしたりしないようご注意ください。
