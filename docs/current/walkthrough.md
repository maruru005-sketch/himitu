# Phase 10: 短期取引向けのおすすめ銘柄（スクリーニング）機能 完了報告

おすすめ銘柄（急騰・出来高急増銘柄）を自動で抽出し、画面上に表示する機能（Phase 10）の実装が完了しました。

## 実施内容

### 1. おすすめ銘柄の抽出ロジック (バックエンド)
- **スクリーニング対象の定義**: `server.py` に、短期トレードで人気があり流動性の高い銘柄群（レーザーテック、ソフトバンクG、三菱重工など約20銘柄）を `SCREENER_UNIVERSE` として定義しました。
- **データ解析とスコアリング**: 新規エンドポイント `GET /api/recommended` を実装しました。このエンドポイントでは、対象銘柄の5分足データを並列で取得し、以下の指標を計算してスコアリングします。
  - **出来高急増**: 直近の出来高と過去の平均出来高の比較。
  - **急上昇**: 当日の始値からの値上がり率。
  - **MACD ゴールデンクロス (GC)**: MACDラインがシグナルラインを下から上に抜けたかどうかの判定。
- スコアが高い順に上位10銘柄をソートしてフロントエンドに返却します。

### 2. おすすめ銘柄表示UI (フロントエンド)
- **タブ切り替え UI の追加**: `index.html` の左側銘柄リスト上部に「監視銘柄」と「おすすめ」を切り替えるタブを実装しました。
- **データ取得と表示**: 「おすすめ」タブを選択すると、バックエンドからおすすめ銘柄のリストを取得し表示します。選出された理由（例: "MACD GC, 急上昇" 等）も銘柄名の下にハイライト表示されるよう工夫しています。
- **シームレスな連携**: 監視銘柄と同様に、おすすめ銘柄リストから銘柄をクリックすることで、即座に右側のチャートエリアに詳細なチャートや各種インジケーターが表示されます。

## 確認手順
1. アプリケーションを起動（`start.bat` 経由）し、ブラウザで画面を開きます。
2. 左側のリスト上部にある「おすすめ」タブをクリックします。
3. 人気銘柄の中から、当日の値動きや出来高に基づいてピックアップされた上位銘柄がリストアップされることを確認します。
4. リスト内の銘柄をクリックし、メインチャートが切り替わることを確認してください。
