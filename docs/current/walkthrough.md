# 実装レポート: AI売買判断と比較機能

Gemini AIを活用して、現在の市場データとニュースから売買判断を行い、既存のテクニカルロジックと比較できる機能を実装しました。

## 実装内容

### 1. バックエンド (`server.py`)
- **AI分析ロジック**: 
  - 直近20本のローソク足データ（OHLCV, VWAP）
  - 最新のテクニカル指標（RSI, BB, MACD）
  - キャッシュされたニュース要約
  - 上記をGemini APIに渡し、「買い・売り・待ち」の判断と具体的な理由、確信度（スコア）を取得するロジックを実装。
- **新規API**: `GET /api/ai-advice/<code>` を追加。ロジックによるシグナルとAIの判断をセットで返します。

### 2. フロントエンド (`index.html`)
- **比較パネル**: 
  - 「AI売買判断比較」カードを右パネルに追加。
  - 「分析」ボタンを押すことでAIの最新見解を取得。
  - ロジック判断（BB+RSI）とAI判断の結果を横並びで表示。
  - AIが挙げた理由と、強気度を示すスコアバーを表示。
- **チャート連携**:
  - AIが「買い」または「売り」と判断した場合、チャートの最新位置に紫色の星印（★）とラベルを表示。

## 使い方
1. 銘柄を選択します。
2. 右パネルの「AI売買判断比較」にある「分析」ボタンを押します。
3. AIがデータを解析し、結果が表示されます。チャート上にもAIの推奨ポイントが表示されます。

## 検証結果
- API連携が正常に行われ、Geminiからの回答をパースしてUIに反映できることを確認しました。
- ロジックが「買い」でもAIが「待ち」と判断するなど、多角的な視点での比較が可能になりました。
