# DayTrade Terminal v3 - Real Data Edition

## ファイル構成
```
daytrade/
├── start.bat     ← Windows 簡単起動スクリプト（ダブルクリックで実行）
├── files/
│   ├── server.py     ← Flask バックエンド（yfinance でリアルデータ取得）
│   ├── index.html    ← フロントエンド（チャート表示・シグナル・UI）
│   ├── config.json   ← 設定ファイル（銘柄リスト、LINEトークン）
│   └── start.ps1     ← (旧) Windows 起動スクリプト
└── docs/             ← ドキュメント・仕様書・実装レポート等
```

## セットアップ

### 必要環境
- Python 3.8 以上
- インターネット接続（Yahoo Finance アクセス用）

### 手動起動（推奨）

```bash
# 1. ライブラリインストール（初回のみ）
pip install flask flask-cors yfinance pandas numpy

# 2. サーバー起動
python server.py

# 3. index.html をブラウザで開く
```

### 簡単起動（Windows 推奨）
1. フォルダ内にある `start.bat` をダブルクリックして実行します。
2. 自動でライブラリが確認・インストールされ、バックエンドサーバーが別ウィンドウで起動します。
3. 数秒後にブラウザが開き、DayTrade Terminal が表示されます。

※サーバー実行中の黒い画面（コマンドプロンプト）は、ツール使用中は閉じないでください。

## API エンドポイント

| エンドポイント | 説明 |
|---|---|
| `GET /api/health` | サーバー疎通確認 |
| `GET /api/stocks` | 全10銘柄の最新価格・トレンドスコア一覧 |
| `GET /api/chart/{code}` | 指定銘柄の1分足OHLCV + BB + RSI + MACD + シグナル |

## 市場時間について
- 東証取引時間: **9:00〜11:30 / 12:30〜15:30**
- 時間外は前日・当日の直近データを使用
- yfinance の1分足データは直近7日分まで取得可能

## フロントエンド機能
- 上昇銘柄TOP10リスト（trendSlope降順）
- 1分足ローソク足チャート + ボリンジャーバンド(20,2σ)
- BUY(赤▲) / SELL(青▼) シグナル矢印（軌跡として残る）
- 右パネル: OHLC・BB値・%B・RSI・出来高
- BUY/SELL ボタン点滅
- シグナルログ（銘柄・時刻・価格）
- 30秒ごとにチャート自動更新
- サーバー未起動時はモックデータで動作

## 最新の追加機能（v3）
- **時間足切り替え**: 1分足から週足まで7種類の時間足に切り替え可能。
- **VWAP・ATR描画**: チャート上にVWAP（出来高加重平均価格）を表示。ATRベースの動的損切り設定にも対応。
- **ポジション管理**: BUY/SELLボタンで仮想ポジションを構築し、リアルタイムにP&L（損益）を計算・表示。
- **バックテスト**: 指定銘柄・時間足の過去300本のデータからシグナル勝率と総損益を計算。
- **通知連携**: 損切り・利確ラインへの接近時に、ブラウザ標準の通知および LINE Notify へのメッセージ送信を実行。
- **設定UI**: ブラウザ上から監視銘柄リストやLINEトークンを編集・保存できる機能。
